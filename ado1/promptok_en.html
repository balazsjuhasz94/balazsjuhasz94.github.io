Prompts Used

This page shows the "question" the model was asked to answer during the automatic categorization of organizations. In simplified terms, the process is similar to asking ChatGPT something — but with a template where different data is substituted in each time, and the results are automatically run and saved. The model is also told exactly what format the answer should be in, which isn't guaranteed, but works reliably in practice.

The Claude Haiku model received the following text for each group of organizations.

Each batch asks the model to correctly categorize 10 organizations. This is because that number of records can still be held in context effectively at once.

{categories_text} was replaced with the list of categories and their descriptions; {orgs_text} was replaced with organization data (name, location, stated goal).

Note: The original categorization was run entirely in Hungarian. The prompts below are English translations provided for reference.

The Categorization Prompt

Categorize the Hungarian civil organizations based on their goals AND location.
Choose EXACTLY ONE category for each organization from the options below.

IMPORTANT RULES:
- Read CAREFULLY each category's description
- "FOCUS" in a category description means this is the PRIMARY activity,
  not just a side aspect
- The "Use if..." section tells you exactly when to choose that category
- The "Does NOT include:" section helps avoid common mistakes
- If uncertain, choose the "other..." category
  within the appropriate main category

{categories_text}

ORGANIZATIONS:
{orgs_text}

ANSWER FORMAT - numbered list only, each line: number. category
IMPORTANT: Write only the category NAME, not the parent category!

Example:
1. dogs
2. high schools in Budapest
3. hospice

Your answer:

Category Descriptions

{categories_text} contained the following categories and their descriptions:

=== CULTURE AND ARTS ===

• arts organizations
  Use if the organization's FOCUS is art, culture.
  Includes: music ensembles, theaters, painting, dance groups,
  cultural events, arts education, galleries, exhibitions.

• ethnic organizations
  Use if the organization's FOCUS is ethnic or national minorities, peoples.
  Includes: Roma, German, Croatian, Serbian minority organizations,
  tradition preservation, folk dance, folk music groups, minority rights.

• religious organizations
  Use if the organization's FOCUS is spreading religious values.
  Includes: church foundations, Bible programs, Christian/Jewish/Islamic
  communities, religious programs, spiritual development.
  Does NOT include: church-run social services (categorize those under social organizations).

• other cultural organizations
  Use if cultural activity, but doesn't fit any specific subcategory.
  Includes: library support, community centers, cultural heritage preservation,
  local history, museum support.

=== HUMAN RIGHTS AND CIVIL SOCIETY ===

• press, media
  Use if the organization's FOCUS is press or media activity.
  Includes: newspapers, online news portals, magazines, radio, television,
  news services, journalism support, media freedom.
  Does NOT include: organizations that merely send newsletters or have a website.

• democracy and transparency
  Use if the organization's FOCUS is strengthening democracy, developing civil society.
  Includes: transparency, anti-corruption, election monitoring,
  civic participation, open government, social dialogue.

• women's rights organizations
  Use if FOCUS is women's rights, gender equality.
  Includes: women's equal opportunities, prevention of violence against women,
  support for mothers, women's career development, women's social roles.

• LGBTQ+ organizations
  Use if FOCUS is LGBTQ+ community rights.
  Includes: lesbian, gay, bisexual, transgender, queer communities,
  gender identity, fighting discrimination based on sexual orientation.

• other human rights organizations
  Use if rights-focused, but not specific to women or LGBTQ+.
  Includes: general human rights organizations, consumer protection,
  workers' rights, tenants' rights, refugee rights, human rights monitoring.

=== EDUCATION ===

• high schools
  Use if supporting a HIGH SCHOOL (foundation/association).

• primary schools
  Use if supporting a PRIMARY SCHOOL (8-year) (foundation/association).

• kindergartens
  Use if supporting a KINDERGARTEN (foundation/association).

• other schools
  Use if supporting OTHER EDUCATIONAL INSTITUTION (vocational school, dormitory, etc.).
  Does NOT include: high schools, primary schools, kindergartens.

• other educational organizations
  Use if education-related, but NOT supporting a specific institution.
  Includes: talent development, student competitions, scholarships, teacher training,
  educational programs, textbooks, pedagogical methods, adult education.

• scientific research
  Use if FOCUS is scientific research, academic activity.
  Does NOT include: support for public education institutions.
